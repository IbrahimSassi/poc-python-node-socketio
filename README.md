Requirements
install https://nodejs.org/en/download/ including npm (package manager)

## Python requirements :
1. pip install -U socketIO-client
2. pip install faker
3. Run python server : Python pyhton/socketio.py


## Node Server requirements :
1. cd node
2. npm install
3. npm run dev:build
4. Run nodejs server : npm run dev

## Vue requirements
1. cd vue-client
2. npm install
3. npm run dev
